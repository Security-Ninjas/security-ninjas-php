Secure File Manager

Mission: Discover and exploit a vulnerability to exfiltrate sensitive data.

  • Hint 1

    The link to a "Secure File Manager" is actually a webapp called "Simple File Manager."

  • Hint 2

    Google for "simple file manager 024 vulnerability".
    Discover the Exploit Database record and follow the directions.

  • Solution

    The default username ('humphrey') is revealed in the login box. Simply visit the URL to:
    /sfm/fm.php?u=humphrey and you're in.
