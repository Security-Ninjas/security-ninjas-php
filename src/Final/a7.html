View Corporate Leadership
Name
Description

Mission: Inject malicious content into the leadership page.

  • Hint 1

    Look at the page source and find the div with id='admin_view'.

  • Solution

    Remove the style 'display:none' in the div with id 'admin_view' to make the hidden div visible.
    You can do this with your browser's developer tools.
    Because there is no server side access control, you can exploit the function.
